Добро пожаловать в установку VapManager!

VapManager — программа для учёта AGV, задач обслуживания, уведомлений и внутреннего чата.

Перед началом работы администратор должен:
1. Развернуть PostgreSQL и базу agv_manager_db.
2. Выдать IP сервера и учётную запись vapmanager.
3. При необходимости выполнить postgresql\configure_remote.ps1 на сервере.

После установки при первом запуске укажите IP сервера БД.
Пароль подключения к базе уже встроен в программу.
