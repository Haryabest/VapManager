ПОЛИТИКА КОНФИДЕНЦИАЛЬНОСТИ VapManager

1. Общие положения
VapManager — корпоративное приложение для учёта AGV и задач обслуживания.
Программа подключается к центральной базе PostgreSQL вашей организации.

2. Какие данные обрабатываются
- учётные записи пользователей (логин, роль, профиль);
- данные об AGV, задачах, уведомлениях и журналах действий;
- сообщения во внутреннем чате между пользователями.

3. Где хранятся данные
Данные хранятся на сервере PostgreSQL, указанном администратором.
Локально приложение может сохранять настройки (config.ini) и файлы журналов.

4. Передача данных
Сообщения чата шифруются перед записью в базу данных.
Пароль подключения к БД задаётся администратором при развёртывании.

5. Права пользователя
Обратитесь к администратору системы для изменения профиля или удаления учётной записи.

6. Контакты
По вопросам обработки данных обращайтесь к администратору VapManager в вашей организации.
